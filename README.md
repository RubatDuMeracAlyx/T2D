# Final semester project with gdx2d 

This project provides several demonstration programs and code examples made using the [gdx2d](https://isc-hei.github.io/gdx2d/) library.
All the demos are destined to be run on a desktop. 
The Javadoc of the gdx2d library [is available online](https://hevs-isi.github.io/gdx2d/javadoc/index.html).

A short video demonstrates the game, [have a look at it here](https://youtu.be/pshk-cnAqoQ). 

## Install

To install in IntelliJ:

1. File > Open, select the `gdx2d-game/` folder
2. 'Project SDK is not defined' banner -> pick any JDK 8+
3. 'Set up Scala SDK' prompt           -> pick any Scala 2.13.14+

## Run the Game

To run the game:
1. Go to `T2D/src/main/scala/ch.hevs.gdx2d.mygame/Menu`
2. press play at the top of the screen
3. Select the number of player and the map and enjoy !

## Keyboard

for Player1:
*WASD* for movements E for Booster and *Q* to reset to the last checkpoint

for Player2:
*IJKL* for movements U for Booster and *O* to reset to the last checkpoint

for Player3:
*TFGH* for movements R for Booster and *Z* to reset to the last checkpoint

for Player4:
*Arrows* for movements *Enter* for Booster and *Shif*t to reset to the last checkpoint

Press Escape anytime to go back to the menu.
